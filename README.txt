Análisis de Datos de MOOCs y Propuesta de KPIs
Este proyecto consta de dos partes: un análisis de bases de datos proporcionadas y una propuesta de KPIs relacionados.
Análisis Exploratorio de Datos (EDA)
En la primera parte del proyecto, se realiza un análisis exploratorio de las bases de datos utilizando Python. El código está disponible en formato de Colab, lo que permite ejecutarlo en cualquier computadora con acceso a Google. Se deben ajustar las rutas de los archivos correspondientes a los dataframes para que funcione correctamente.
El análisis se centra en la generación de gráficas de correlación para cada dataset, lo cual nos proporciona información clave para la segunda parte del proyecto.
Propuesta de KPIs
En la segunda parte del proyecto, se presenta una propuesta de KPIs con el objetivo de alcanzar las siguientes metas:
1. Aumento de Ganancias: El KPI propuesto es vigilar el aumento anual de ganancias, estableciendo una meta mínima de 90 millones. Este indicador nos permitirá evaluar el desempeño financiero del proyecto.
2. Aumento del Alcance de Alumnos Inscritos: El KPI propuesto es vigilar el aumento anual de alumnos inscritos, con una meta anual de un millón de alumnos. Este indicador nos ayudará a medir la efectividad de las estrategias de captación de estudiantes.
3. Aumento de la Producción: El KPI propuesto es medir la cantidad anual de cursos producidos, estableciendo una meta mínima de 500 cursos por año. Este indicador nos permitirá evaluar la capacidad de producción y la variedad de cursos ofrecidos.
Presentación del Dashboard
La propuesta de KPIs se presenta en forma de un Dashboard interactivo, utilizando Power BI. Para acceder a la presentación, es necesario abrir el archivo en Power BI para explorar los elementos más relevantes de la tabla de Udemy.
Instrucciones
1. Ejecuta el código del análisis exploratorio de datos en un entorno de Colab, asegurándote de ajustar las rutas de los archivos correspondientes.
2. Abre el archivo del Dashboard en Power BI para explorar los KPIs propuestos y obtener una visión más completa de los resultados del análisis.
